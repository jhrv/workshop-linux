find

Søk etter filer og mapper på filsystemet


find . -name fil # finner filer og mapper med navn 'fil' i fra katalogen du står i

find src -name *test* -type d # finner mapper som inneholder 'test' i navnet i katalogen src

grep

Søker etter innhold i filer 


grep -i img index.html # finner linjene med "img" i index.html (i = ignore case)

grep -r img src # finner linjene med img i alle filene i src (rekursivt)

cut

splitter innhold basert på delimiter


cut -d '=' -f 1 # deler output på '=' og printer første kolonne

cut -d '.' -f 2 # deler output på '.' og printer andre kolonne

(krever innhold på stdin)

sed

stream editor, s & r


sed -i 's/hei/halla/g' fil.txt # bytter ut alle hei med halla i 'fil.txt'

sed 's/hei/halla/g' # printer stdin der hei er byttet ut med halla (krever innhold på stdin)

xargs - for each

wc - word/line/char count

uniq - kun unike verdier i lista
